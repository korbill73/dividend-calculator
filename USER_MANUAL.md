## 📝 개요

이 대시보드는 개인 투자자가 배당 포트폴리오를 관리하고, 장기 자산 성장을 시뮬레이션할 수 있는 통합 플랫폼입니다.

### 주요 기능
- 📈 **배당 포트폴리오 관리**: 월별 배당금 추적 및 시각화
- 🎲 **자산 성장 시뮬레이션**: 다양한 수익률 시나리오 비교
- 💾 **데이터 백업**: 로그인 시 자동 저장 및 동기화
- 📊 **상세한 분석**: 배당 수익률, 총 자산, 연간 배당 등

## 💰 배당 포트폴리오 관리

### 3.1 포트폴리오 개요 확인

**경로**: 메뉴 > Dividends

**표시 정보:**
1. **Total Assets (총 자산)**
   - 보유 종목의 총 평가액
   - 계산식: Σ (수량 × 현재가)

2. **Annual Dividends (연간 배당)**
   - 선택한 연도의 총 배당금
   - 월별 배당금 합계

3. **Dividend Yield (배당 수익률)**
   - 실현 수익률
   - 계산식: (연간 배당 / 총 자산) × 100

### 3.2 종목 추가하기 (로그인 필요)

**단계:**
1. "Add Stock (종목 추가)" 버튼 클릭
2. 종목 정보 입력:
   - **종목명**: ETF/주식 이름
   - **티커**: (선택사항)
   - **수량 (Qty)**: 보유 주식 수
   - **가격 (Price)**: 현재 주가
   - **배당일 (Day)**: 월 배당일 (1-31)
   - **수익률 (Yield)**: 배당 수익률 (%)

3. 월별 배당금 입력:
   - 각 월의 셀을 클릭하여 배당금 입력
   - 숫자만 입력 (쉼표 자동 추가)
   - 예: "450000" 입력 시 "450,000" 표시

### 3.3 종목 수정하기

**방법:**
- **종목명 변경**: 상단 종목명 셀 클릭 후 입력
- **수량/가격**: Info 열의 Qty/Price 입력란 수정
- **배당일/수익률**: Info 열의 Day/Yield 입력란 수정
- **월별 배당금**: 해당 월 셀 클릭 후 수정

**실시간 계산:**
- 입력 즉시 총액 자동 계산
- 연간 합계 자동 업데이트

### 3.4 종목 삭제하기

**방법:**
1. 종목명 위에 마우스 오버
2. 우측 상단 🗑️ (휴지통) 아이콘 클릭
3. 종목 즉시 삭제

⚠️ **주의**: 삭제 시 복구 불가능 (로그인 사용자는 Supabase에서 복원 가능)

### 3.5 연도별 데이터 관리

**기능:**
- ← → 버튼으로 연도 이동
- 연도별 독립적인 배당 데이터
- 과거 데이터 비교 및 분석

**활용 예:**
```
2020년: 실제 배당금 기록
2021년: 실제 배당금 기록
2022년: 실제 배당금 기록
2023년: 실제 배당금 기록
2024년: 현재 배당금
2025년: 예상 배당금
```

### 3.6 배당 매트릭스 읽는 법

**테이블 구조:**
```
┌─────────────┬────────┬────────┬────────┐
│ 월별 총액    │ 종목1  │ 종목2  │ 종목3  │
├─────────────┼────────┼────────┼────────┤
│ 1월: 500,000│ 200,000│ 150,000│ 150,000│
│ 2월: 450,000│ 180,000│ 140,000│ 130,000│
...
└─────────────┴────────┴────────┴────────┘
      ↑            ↑        ↑        ↑
   월별합계    종목1배당  종목2배당  종목3배당
```

**색상 의미:**
- 🟢 **초록색**: 배당이 있는 월
- ⚫ **회색**: 배당이 없는 월
- 🟡 **노란색**: 총액 행

## 🎲 시뮬레이션 사용법

### 4.1 시뮬레이션 개요

**경로**: 메뉴 > Simulation

**목적**: 장기 자산 성장 예측 및 시나리오 비교

### 4.2 설정 패널 (좌측)

#### 1) Simulation Period (시뮬레이션 기간)

**설정 항목:**
- **Start Year (시작년도)**: 시뮬레이션 시작 연도
- **End Year (종료년도)**: 시뮬레이션 종료 연도

**권장 설정:**
- 일반적으로 20-30년 기간 설정
- 예: 2025년 ~ 2050년 (25년)

#### 2) Target Annual Returns (목표 수익률)

**3가지 시나리오:**

**Conservative (보수적)**: 5-7%
- 안정적 투자
- 채권 중심 포트폴리오
- 낮은 변동성

**Moderate (중립적)**: 8-10%
- 균형 포트폴리오
- 주식 60% + 채권 40%
- 역사적 평균 수익률

**Aggressive (공격적)**: 12-15%
- 고수익 추구
- 주식 80% + 성장주
- 높은 변동성

**권장 값:**
```
보수적: 5%
중립적: 8%
공격적: 12%
```

#### 3) Monthly Contribution (월 추가 불입금)

**입력 방법:**
- 만원 단위로 입력
- 예: "100" 입력 = 100만원 = 1,000,000원

**활용 팁:**
- 매월 정기 투자금 입력
- 급여의 20-30% 권장
- 무리하지 않는 범위 설정

#### 4) Current Account Balances (현재 계좌 잔액)

**계좌 종류 (수정 가능):**
- **계좌명**: 클릭하여 원하는 이름으로 변경
- **잔액**: 만원 단위 입력

**일반적인 계좌 구성:**
```
IRP (개인형 퇴직연금)
ISA (개인종합자산관리계좌)
연금저축
일반 계좌
```

**총합 표시:**
- 하단에 전체 계좌 합계 자동 계산
- 파란색으로 강조 표시

### 4.3 그래프 분석 (우측)

**차트 읽기:**

**4개 선:**
1. 🔴 **Conservative (빨간 점선)**
   - 보수적 시나리오
   - 하한선 참고

2. 🔵 **Moderate (파란 점선)**
   - 중립적 시나리오
   - 기준선 (가장 현실적)

3. 🟢 **Aggressive (초록 점선)**
   - 공격적 시나리오
   - 상한선 참고

4. 🟡 **Actual (노란 실선)**
   - 실제 투자 성과
   - 입력한 데이터 기반

**Y축**: 억 단위 표시
**X축**: 연도-월

### 4.4 실제 성과 입력

**테이블 사용법:**
1. 하단 "Detailed Projection Data" 테이블 확인
2. **Actual Result** 열에서 해당 월 클릭
3. 실제 자산 가치 입력
4. Enter 또는 다른 셀 클릭 시 저장

**입력 예:**
```
2025-01: 53,500,000원
2025-02: 54,800,000원
2025-03: 55,200,000원
...
```

**Difference 열:**
- 자동 계산: Actual - Moderate
- 🟢 초록색: 목표보다 높음
- 🔴 빨간색: 목표보다 낮음

### 4.5 시뮬레이션 활용 전략

**1단계: 초기 설정**
```
✓ 현재 자산 입력
✓ 월 불입금 설정
✓ 목표 연도 설정
```

**2단계: 시나리오 분석**
```
✓ 3가지 수익률 비교
✓ 그래프에서 차이 확인
✓ 최종 자산 예측
```

**3단계: 실제 성과 추적**
```
✓ 매월 실제 값 입력
✓ 목표와 비교
✓ 전략 조정
```

**4단계: 정기 리뷰**
```
✓ 분기별 성과 검토
✓ 불입금 조정
✓ 목표 재설정
```

## ❓ FAQ

### Q1: 샘플 데이터를 내 데이터로 바꿀 수 있나요?
**A:** 로그인하시면 모든 샘플 데이터를 삭제하고 본인의 데이터를 입력하실 수 있습니다.

### Q2: 데이터가 사라질까 걱정됩니다.
**A:** 로그인 사용자는 모든 데이터가 Supabase에 자동 저장됩니다. 브라우저 캐시가 삭제되어도 로그인하면 복원됩니다.

### Q3: 모바일에서도 사용 가능한가요?
**A:** 네, 반응형 디자인으로 모바일/태블릿에서도 사용 가능합니다. 다만 배당 매트릭스는 가로 스크롤이 필요할 수 있습니다.

### Q4: 실제 API와 연동되나요?
**A:** 현재는 수동 입력 방식입니다. 향후 증권사 API 연동을 고려 중입니다.

### Q5: 여러 계정을 관리할 수 있나요?
**A:** 하나의 Google 계정당 하나의 포트폴리오를 관리할 수 있습니다. 여러 계정은 다른 Google 계정으로 로그인하세요.

### Q6: 배당금은 세전인가요, 세후인가요?
**A:** 사용자가 직접 입력하는 방식이므로, 세전/세후를 선택하여 입력하시면 됩니다. 일관성 유지를 권장합니다.

### Q7: 시뮬레이션이 정확한가요?
**A:** 시뮬레이션은 단순 복리 계산 기반입니다. 실제 투자는 변동성, 세금, 수수료 등 다양한 요인의 영향을 받습니다. 참고용으로만 사용하세요.

### Q8: 배당금을 자동으로 계산할 수 있나요?
**A:** 현재는 수동 입력입니다. 정확한 배당 내역을 위해 증권사 거래 내역을 확인하여 입력하세요.

### Q9: 종목을 몇 개까지 추가할 수 있나요?
**A:** 이론상 무제한이지만, 화면 표시를 고려하여 10-20개 이내를 권장합니다.

### Q10: 데이터 보안은 어떻게 되나요?
**A:** Supabase의 Row Level Security (RLS)를 통해 본인의 데이터만 접근 가능합니다. Google OAuth 인증으로 보안이 강화됩니다.
